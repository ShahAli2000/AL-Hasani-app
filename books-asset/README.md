# Books Asset

PDF files for the AL-Hasani app library.

## Structure

Each subfolder corresponds to a category:
- `quran/` — Quran & Tafsir
- `hadith/` — Hadith collections
- `dua/` — Dua & Ziyarat
- `fiqh/` — Jurisprudence
- `aqeedah/` — Theology
- `history/` — Islamic history
- `ethics/` — Ethics & morality
- `biography/` — Biography

## URL Pattern

Raw file access:
`https://raw.githubusercontent.com/ShahAli2000/AL-Hasani-app/main/books-asset/{category}/{book-id}.pdf`
